자바스크립트에서 사용할 수 있는 모든 값은 프로퍼티 값이 될 수 있다. 프로퍼티 값이 함수일 경우, 일반 함수와 구분하기 위해 메서드라고 부른다 객체 내부에서 객체의 프로퍼티를 참조하고 조작할 수 있는 동작을 메서드라 부른다.

즉 메서드는 객체에 묶여 있는 함수를 의미한다.

```js
var person = {
  name: "Lee",
  age: 20,
  hello: function () {
    console.log("hello :" + this.name);
  },
};
```
